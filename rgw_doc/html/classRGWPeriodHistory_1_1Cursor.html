Detailed Description

Cursor tracks a position in the period history and allows forward and backward traversal. Only periods that are fully connected to the current_period are reachable via a Cursor, because other histories are temporary and can be merged away. Cursors to periods in disjoint histories, as provided by insert() or lookup(), are therefore invalid and their operator bool() will return false.
