Get data about an object out of RADOS and into memory. bucket: name of the bucket the object is in. obj: name/key of the object to read data: if get_data==true, this pointer will be set to an address containing the object's data/value ofs: the offset of the object to read from end: the point in the object to stop reading attrs: if non-NULL, the pointed-to map will contain all the attrs of the object when this function returns mod_ptr: if non-NULL, compares the object's mtime to *mod_ptr, and if mtime is smaller it fails. unmod_ptr: if non-NULL, compares the object's mtime to *unmod_ptr, and if mtime is >= it fails. if_match/nomatch: if non-NULL, compares the object's etag attr to the string and, if it doesn't/does match, fails out. get_data: if true, the object's data/value will be read out, otherwise not err: Many errors will result in this structure being filled with extra informatin on the error. Returns: -ERR# on failure, otherwise (if get_data==true) length of read data, (if get_data==false) length of the object
