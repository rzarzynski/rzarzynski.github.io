void librados::ObjectOperation::omap_cmp ( const std::map< std::string, std::pair< bufferlist, int > > &  assertions,
int *  prval 
)

get key/value pairs for specified keys

Parameters
assertions[in] comparison assertions
prval[out] place error code in prval upon completion

assertions has the form of mappings from keys to (comparison rval, assertion) The assertion field may be CEPH_OSD_CMPXATTR_OP_[GT|LT|EQ].

That is, to assert that the value at key 'foo' is greater than 'bar':

ObjectReadOperation op; int r; map<string, pair<bufferlist, int> > assertions; bufferlist bar(string('bar')); assertions['foo'] = make_pair(bar, CEPH_OSD_CMP_XATTR_OP_GT); op.omap_cmp(assertions, &r);
